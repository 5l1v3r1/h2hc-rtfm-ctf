# Bank Network Tool

## Description
It's a php web application, where the objective is get a shell on the web server, abusing of the vulnerable functionalities.

## Dependencies
* docker
* docker-compose

## How to 

| Command       | Action        |
| ------------- |:-------------:|
| make          | build the app |
| make run      | start the app |
| make stop     | stop the app  |
| make stop     | stop the app  |
| make delete   | delete the app|
