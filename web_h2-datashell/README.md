# H2DataShell

## Description
It's h2 database instance with a WAF in front of it, the attacker need to bypass the WAF.

## Dependencies
* docker
* docker-compose

## How to 

| Command       | Action        |
| ------------- |:-------------:|
| make          | build the app |
| make run      | start the app |
| make stop     | stop the app  |
| make stop     | stop the app  |
| make delete   | delete the app|
